
# Quiz System - Python Flask Application

An interactive quiz system built with Flask (Python backend) and vanilla JavaScript for the frontend.

## Features

- 📝 Multiple choice questions
- ⏱️ Timer for each question
- 📊 Score tracking
- 🎯 Difficulty levels
- 🔄 Question navigation
- 📱 Responsive design

## Installation

### Prerequisites
- Python 3.7+
- pip

### Setup Instructions

1. Clone the repository
2. Create virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
5. Open: `http://127.0.0.1:5000`

## Tech Stack
- Backend: Flask (Python)
- Frontend: HTML5, CSS3, JavaScript
- Templating: Jinja2

## Project Structure
- app.py - Main Flask application
- questions.py - Quiz questions data
- static/style.css - CSS styles
- templates/index.html - Quiz interface
